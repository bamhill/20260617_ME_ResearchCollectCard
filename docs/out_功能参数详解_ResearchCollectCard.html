ResearchCollectCard · 功能参数详解

CLI 全参数 · Source 配置 · Fetcher 接口 · 输出格式

一、CLI 命令全参数

paper — 论文日报

参数类型默认说明
--datestr今天日期 YYYY-MM-DD
--fromstr—起始日期，与 --to 配用
--tostr—结束日期，多天自动合并去重
--sourcestrtoptop(顶会6) | cs(2) | journal(51) | all(59)
--topicstr—多标签AND筛选，空格分隔。例: "ABS4 管理信息系统"
--translateflagFalse翻译英文摘要为中文 (DeepSeek)

wechat — 公众号日报

参数类型默认说明
--datestr今天日期 YYYY-MM-DD
--fromstr—起始日期
--tostr—结束日期

管理命令

命令说明
python cli.py sources列出所有66源 (名称+类型+查询参数)
python cli.py cards列出已生成卡片 (ID+日期+条目数)
python cli.py build重建 out/index.html dashboard

二、Source 配置参数

字段类型必填说明
idstr✓唯一标识，用于去重和筛选
namestr✓显示名称，出现在卡片分区标题
typeenum✓arxiv | crossref | rss | biorxiv
querystr—arXiv: cat查询 · CrossRef: ISSN · RSS: URL
tagslist[str]—标签: 顶会/期刊+分级+领域。用于筛选和归类

三、Fetcher 策略模块

📡
ArxivFetcher
接口: arXiv API
source_type: "arxiv"
query格式: cat:cs.LG AND all:"ICML"
日期: API过滤
📚
CrossrefFetcher
接口: CrossRef API
source_type: "crossref"
query格式: 0950-7051(ISSN)
日期: from-pub-date过滤
📰
RssFetcher
接口: feedparser
source_type: "rss"
query格式: RSS/Atom URL
日期: published_parsed / updated_parsed / summary回落
过滤: TOC等元条目自动跳过
🧬
BiorxivFetcher
接口: bioRxiv API
source_type: "biorxiv"
query格式: (不使用)
日期: API日期参数

四、热点提取模块

YAKE 关键词提取

无监督关键词提取算法
从全量摘要中提取关键短语
中英文通用 · 离线运行

DeepSeek 翻译分类

将关键词翻译为中文
分为"研究主题"和"技术方法"两类
需 DEEPSEEK_API_KEY

五、输出格式

产物路径模式格式大小
论文卡out/paper_{source}_{topic}_{date}.html自包含HTML~300KB
公众号卡out/wechat_{date}.html自包含HTML~280KB
Dashboardout/index.html卡片浏览面板~570KB
卡片JSONdata/cards/{card_id}.json结构化数据~50KB
设计约束： HTML零CDN依赖 · file://直开 · CSS变量控制主题 · 对比度≥5.3:1 · Jinja2模板渲染 · ensure_ascii=False JSON编码
