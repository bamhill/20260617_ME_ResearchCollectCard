三、命令行参考

# 论文日报 python cli.py paper # 顶会日报 (6源) python cli.py paper --source journal # 期刊精选 (51源) python cli.py paper --source journal --topic CAS1 可解释性 # 交叉筛选 python cli.py paper --from 2026-06-10 --to 2026-06-18 --source journal # 公众号日报 python cli.py wechat # 今日7源 # 管理 python cli.py sources # 列出所有源 python cli.py build # 重建dashboard python -m pytest tests/ -q # 46 tests

四、新增来源

+ CrossRef 期刊

编辑 sources.json
{"type":"crossref"}
"query":"ISSN号"
加 tags 标签

+ RSS 直连

编辑 sources.json
{"type":"rss"}
"query":"RSS_URL"
自动归入 journal_sources()

五、输出路径

类型路径格式
论文卡out/paper_{source}_{topic}_{date}.html杂志风HTML
公众号卡out/wechat_{date}.html杂志风HTML
Dashboardout/index.html卡片浏览面板
卡片数据data/cards/*.json结构化JSON
注意： out/ 目录已纳入版本控制，生成的卡片可直接在GitHub预览。data/cards/ 不提交（gitignore）。
