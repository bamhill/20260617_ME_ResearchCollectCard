ResearchCollectCard · 系统总览

每日科研新资讯卡片生成器 — 自动化文献追踪，杂志风简报输出

项目定位

ResearchCollectCard 是一套轻量级、数据驱动、零部署的科研文献日报工具。将 arXiv 顶会追踪、CrossRef 期刊查询、期刊 RSS 直连和公众号四路数据源整合为单页杂志风 HTML 卡片，双击即开，一键存图。

🎯 解决问题

高校课题组的真实日常：每天手动刷 arXiv + 期刊主页 + 公众号，至少 1.5 小时。自动聚合 + 热点提取 + 排版输出，压缩到 3 分钟。

✨ 核心特色

四路多源聚合 · YAKE+LLM热点提取 · 杂志风视觉设计 · 多标签交叉筛选 · file://直开零CDN

日常使用流程

Step 1
启动 WeRSS
docker start wewe-rss
