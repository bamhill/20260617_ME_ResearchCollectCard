# 小红书帖子

## 标题
实验室日常：用 Claude Code 做了个文献追踪小工具

## 正文

在高校做研究，不管是带学生还是自己跟方向，每周新论文都看不过来。arXiv 好几个分类、手上 30 多本对口期刊、再加几个公众号——光刷新就花不少时间。

这学期试了一下用 Claude Code 搭个小工具，帮我自动做这件事。用了一阵觉得还行，分享一下思路，也听听大家的意见。

大概是这样：
- 每天自动抓 arXiv 顶会论文（ICML/NeurIPS/ICLR 等）、CrossRef 期刊新文章、几个公众号推送
- 从摘要里自动提几个"今天的热点词"，分两类：研究主题在讨论什么、技术方法在用什么
- 输出是一页杂志风的简报，米色纸张那种老派排版，可以截图发课题组群，也可以自己归档

🔗 代码放 GitHub 了，有需要的老师和同学自取：bamhill/20260617_ME_ResearchCollectCard

还在完善中——期刊 RSS 直连还没完全调通，部分期刊延迟比较大。欢迎一起提意见。
