# 踩坑记录

## CLI

### `_filter_by_topic` 多词误杀
- 日期：2026-06-19
- 现象：`--topic "ABS4 管理信息系统"` 返回空卡
- 根因：原逻辑 `topic in tags` 把完整字符串当单个 tag 匹配
- 修复：`topic.split()` → `all(t in tags for t in topics)`
- 文件：src/pipeline/transform.py

### `add_source` 不认 crossref
- 日期：2026-06-19
- 现象：新增期刊源后 `add_source` 误存入 wechat_sources
- 根因：只路由 arxiv/biorxiv → paper_sources
- 修复：加 `"crossref"` 到路由条件
- 文件：src/config.py

## 数据

### hotspots 模型变更致保存遗漏
