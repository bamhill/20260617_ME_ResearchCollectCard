# 通用技巧

## Python 替代 Edit 处理中文文件
- Python `c = open(f).read(); c.replace(old, new); open(f,'w').write(c)` 稳定可靠
- Edit 工具含中文文件反复编码冲突

## gstack browse 路径大小写
- Windows 下 `D:/` 和 `d:/` 有区别，截图路径用大写

## skills.json 数据驱动
- 新增来源只改 `sources.json`，不改代码
- 格式：`{"id","name","type","query","tags"}`

## CLI 多标签 AND
- `--topic "A B C"` → `topic.split()` → `all(t in tags for t in topics)`
